# MatchVision AI

MatchVision AI is a public, frontend-only SaaS demonstration for **AI football video analytics**, **soccer performance analysis software**, match review, player metrics, tactical phases, clips, and coaching reports. It was created by Ahsan Inam for XsolAI as a portfolio and product-discovery asset.

## Included

- SEO-focused landing and pricing pages
- Local demo login, signup, forgot-password, checkout, and onboarding flows
- Match center with synthetic football footage and event-indexed playback controls
- Deterministic momentum, possession, player map, player ranking, and team metrics
- Event filtering, phase selection, clip creation, player search, tactical analysis, report builder, settings, and billing
- Generated football imagery and dashboard reference
- Responsive desktop and mobile layouts
- Metadata, canonical URLs, Open Graph cards, JSON-LD, sitemap, and robots rules
- Browser QA evidence and passed same-viewport visual comparison

## Demo notice

This repository contains no computer-vision backend, match upload pipeline, tracking system, real player data, authentication provider, payment processor, or private project identifier. Every club, player, competition, fixture, event, tracking point, metric, clip, report, integration, and subscription is fictional and deterministic. Form data is never transmitted.


## Architecture

- Next.js App Router and TypeScript
- React Server Components for route shells
- Isolated Client Components for local interactions
- Recharts for deterministic sports visualizations
- Phosphor Icons for interface iconography
- Generated raster assets for synthetic footage
- `localStorage` for demo persistence
